# 🚀 Sistema de Análise para Aluguel

> Projeto desenvolvido em Java com foco na prática de Programação Orientada a Objetos (POO), lógica de programação e boas práticas de desenvolvimento.

---

## 📖 Sobre o Projeto

Este projeto simula um sistema que analisa se uma pessoa pode alugar um imóvel com base em informações fornecidas pelo usuário.

O sistema solicita:

- 👤 Nome
- 🎂 Idade
- 💰 Renda mensal
- 🏠 Valor do aluguel

Após receber os dados, realiza a análise e informa se a pessoa está apta ou não para alugar o imóvel.

---

## ✨ Funcionalidades

- Cadastro de pessoas
- Validação de dados
- Análise da capacidade financeira
- Aplicação de Programação Orientada a Objetos
- Separação em pacotes (`application` e `entities`)
